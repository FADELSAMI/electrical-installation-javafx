# Electrical Installation Manager

A desktop application developed in JavaFX as part of a "Application development" Bachelor's degree project.

## Overview

This application allows users to manage a residential electrical installation by creating rooms, assigning them to floor plans, and placing electrical devices using a graphical drag-and-drop interface.

The project follows the DAO (Data Access Object) design pattern and uses a Firebird relational database for data persistence.

## Features

- Create and manage electrical installations
- Manage rooms and room types
- Manage floor plans
- Associate rooms with floor plans
- Associate electrical devices with rooms
- Drag & Drop device placement
- Device rotation and repositioning
- Persistent storage of device locations
- Unit tests with JUnit

## Technologies

- Java
- JavaFX
- Firebird SQL
- JDBC
- DAO Pattern
- MVC Architecture
- JUnit 5
- Git / GitHub
